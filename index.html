CP Roadmap

Get the basics right

Make sure you enter CP after you have a decent grip in the programming language you are using. If you use C++ STL comes handy while coding. After you are proficient in a language, get your implementations right.
Be it any program, make sure you implement it correctly, you don't want to be in a situation where you get the logic but you can't implement it. Understand the concepts like Time Complexity and Space Complexity. Search for problems with implementation tag.
Some sample questions would be : Grade Allocation, Filling Diamonds, Excersing Walk, SALARY, CSUB, CNOTE

Start implementing basic DS & Algo

After getting a raw idea on the basic Data Strucutres and Algorithms, implement these on some questions of your choice in any coding platform like Codeforces / CodeChef. Some Important topics are Sorting, Greedy, binary search and fast exponentiation don't forget to checkout counting sort. Search for problems with a DS or Algo tag like Queue or Binary Search, etc;
Some sample questions would be : ABCDEF, LOWSUM, Alternating Current, Constant Palindrome Sum.

Dive deep into advanced DS & Algo

If you feel you are confident with the basic DS and Algo, then you might as well look into the more advanced DS and Algorithms like Segment Trees, Disjoint Set Union, Fenwick Tree, Graph Algorithms, DP,etc;

Become an expert in DSA

This is the great leap in the domain of CP, where you have profound knowledge of the existing topics and are ready to apply this knowledge to any new / unseen problem. hard Graph Algorithms are a challenge to take on. This is where you become one of the finest.
Some topics to study would be : DP Optimizations (eg: Convex-Hull Trick, Divide and Conquer Optimization, Knuth Optimization), Fast Fourier Transform, Game Theory, Advanced String Algos like KMP, Rabin Karp, Z-Function, Flows ( Max-Flow, Min Cost Max Flow ),etc;
